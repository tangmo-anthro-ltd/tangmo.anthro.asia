Vorny Note

If you’ve been following me for a while, it is no secret that I am into vore and muscles. Specifically, I love being the prey in soft-vore with ideally both party having a muscular build. Goo-ey digestion is the furthest I’d go. Thus, this character is intended to be a prey, but with a fun gimmick.

When the torso area is wet, it can release some ink with the same colour as its fur pattern. If on a pred’s tongue, this ink would leave temporary mark, similar to colour from a lollipop or shush. Passerby could recognize the colour pattern and know who is inside. Despite the look, the body is not melting.

How it worksThis trait was supposed to be a defense mechanism, similar that of a squid. However, the evolution has not caught up, leaving him with this ineffective power. The ink secretion is voluntary, but when startled or scared, it may be unintentionally leaked by instinct. Since the dye can only be released when wet, this wouldn’t be a problem day-to-day as the body would be dry.

Despite intending to be a prey, a few friends have already mentioned to that this character look more like a pred. In fact, I do have a super-power idea for a pred character. If you’re interested, let’s have a chat!
